> **A Next-Gen Futuristic WhatsApp Bot**

---

## 🌐 Run Locally

Clone the project

```bash
  git clone https://github.com/Claretiano/Xana-MD
```

Go to the project directory

```bash
  cd Xana-MD
```

Install dependencies

```bash
  npm install
```

Install dependencies **(recommended)**
```bash
 npm install --force
```

Start the server

```bash
  npm start
```
